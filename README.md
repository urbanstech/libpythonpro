Suportada versão 3 de Python

Para instalar:

'''console
mkdir libpythonpro/
python3 -m venv .venv
source ./ven/bin/activate
pip install -r requirements-dev.txt
'''

Para conferir qualidade de código:

'''console
flake8 
'''
